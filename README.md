# Дачный советник — тестовая версия

Минимальный сайт, на котором пользователь:

- описывает проблему растения;
- указывает культуру, регион и место выращивания;
- при желании загружает фотографию;
- получает ответ от OpenAI через Responses API.

В этой версии нет регистрации, оплаты, дневных ограничений и базы данных.
Запросы не сохраняются приложением.

## 1. Что понадобится

- Python 3.10 или новее;
- API-ключ OpenAI;
- интернет на компьютере или сервере, где запущен сайт.

Важно: подписка ChatGPT Plus не включает расходы OpenAI API. API оплачивается отдельно в аккаунте OpenAI Platform.

## 2. Установка на Windows

Откройте PowerShell в папке проекта и выполните:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env
notepad .env
```

В файле `.env` замените значение `OPENAI_API_KEY` на настоящий ключ.

Запуск:

```powershell
uvicorn app:app --reload
```

Откройте в браузере:

```text
http://127.0.0.1:8000
```

## 3. Установка на macOS или Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
nano .env
uvicorn app:app --reload
```

## 4. Настройка модели

По умолчанию используется:

```env
OPENAI_MODEL=gpt-5-mini
```

Модель принимает текст и изображения. Её можно заменить в `.env`, не меняя код сайта.

## 5. Безопасность

- Никогда не вставляйте API-ключ в `static/app.js` или HTML.
- Не публикуйте файл `.env`.
- Перед публичным запуском добавьте ограничения запросов, защиту от ботов,
  журналирование расходов, юридические документы и нормальную авторизацию.
- Текущая версия предназначена только для закрытого теста.

## 6. Проверка состояния

После запуска откройте:

```text
http://127.0.0.1:8000/api/health
```

Поле `api_key_configured` покажет, видит ли приложение ключ.

## 7. Структура

```text
app.py                 сервер и обращение к OpenAI
static/index.html      интерфейс
static/styles.css      оформление
static/app.js          отправка формы и вывод ответа
.env.example           пример настроек
requirements.txt       зависимости Python
```
